# Cisco Packet Tracer – Basic Network Configuration

##  Description

This project demonstrates basic network configuration using Cisco Packet Tracer.

## ⚙️Configurations Done

* Changed device hostnames
* Assigned IP addresses to devices
* Configured router/switch interfaces
* Verified connectivity using ping

##  Devices Used

* Routers
* Switches
* PCs

##  Network Features

* Proper IP addressing scheme
* Communication between devices
* Basic network setup

##  File Included

* Packet Tracer file (.pka)

##  Testing

All devices were tested using ping and are successfully communicating.
